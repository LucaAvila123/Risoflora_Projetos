# Atualização dos Processos Gerenciais
**Cliente**: equipe de voluntários, principalmente de Secretaria, Estrutura e Financeiro

**Proposta**: automatização dos processos de envio de mensagem, elaboração de material de educação digital, orientações gerais para facilitação dos processos do cursinho e criação de um painel digital para observação de dados do cursinho popular

**Justificativa**: alguns processos não podem funcionar de maneira arcaica porque demanda tempo e corrigir certas falhas é muito mais fácil quando uma tecnologia, simples que seja de aprender, é utilizada de maneira devida; soma a isso também observação de dados como um fator crucial para planejamentos futuros

**Requisitos**: a base de dados (tornados sigilosos) dos estudantes do cursinho, um resumo de todos os processos gerenciais para saber como incrementá-los e acesso aos documentos de planejamento

**Carga horária prevista**: 20 horas

***Acompanhamento***

Ferramenta para envio automático de mensagens, usando pywhatkit: 2h
