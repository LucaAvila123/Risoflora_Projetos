# MOODLE: é uma proposta razoável?

Moodle é um sistema de gestão de aprendizagem (LMS), utilizado por uma série de universidades públicas no Brasil como uma solução EaD de código aberto

Não é um substituto, de forma alguma, ao ensino presencial, no entanto, é uma plataforma muito comum de uso cotidiano por uma série de funcionalidades que tem, podendo ser uma alternativa viável para centralização de contatos e de materiais do cursinho popular Risoflora (é necessário estudar como isso seria aplicável)
