# Plataforma de Simulados

**Cliente**: professores do cursinho

**Proposta**: plataforma digital para elaboração de simulados futuros que consiga avaliar as disciplinas e as categorias de diferentes questões e consiga fazer um simulado sozinha para aplicação

**Justificativa**: facilitar elaboração de provas por parte dos professores do cursinho e também possibilitar elaboração de um projeto futuro de elaboração de simulados

**Requisitos**: um acervo de questões de todas as matérias possíveis, um modelo treinável para identificação das disciplinas das questões e classificação de seus graus de dificuldade e uma interface para entrega de simulados completos

**Carga horária prevista**: 40h


Etapas do Projeto:
1- Obtenção de uma base de dados do ENEM que pode ficar disponível offline; uso da API do ENEM a partir de opensource para verificação de possibilidade: https://docs.enem.dev/introduction
